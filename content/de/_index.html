MEDIENÖKOLOGISCHE INFRASTRUKTUREN FÜR BIODIVERSITÄT

Interdisziplinäres Designforschungs­projekt (SNF, 2020-2024)

{{< blocks/link-down color="yellow" >}}
{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} Wie können mediale Design-In­terventionen und Infrastrukturen auf Grundlage des Internets der Dinge ökologisch und kulturell zur Förderung von Biodiversität in lokalen Ökosystemen beitragen? {{< figure src="images/mitwelten_logo.png" >}} {{% /blocks/lead %}} {{< blocks/section color="info" >}} {{% blocks/feature icon="fa-map" title="Merian Gärten" %}} Feldstudie in Basel/Münchenstein BL, 2021 {{< figure src="images/merian.png" text="Merian Gärten" link="https://www.mitwelten.org/fs1" width="90%" >}} {{% /blocks/feature %}} {{% blocks/feature icon="fa-map" title="Dreispitz Areal" %}} Feldstudie in Basel/Münchenstein BL, 2022 {{< figure src="images/dreispitz.png" text="Dreispitz Areal" link="https://www.mitwelten.org/fs2" width="90%" >}} {{% /blocks/feature %}} {{% blocks/feature icon="fa-map" title="Reinacher Heide" %}} Feldstudie in Reinach BL, 2023 {{< figure src="images/reinach.png" text="Reinacher Heide" link="https://www.mitwelten.org/fs3" width="90%" >}} {{% /blocks/feature %}} {{< /blocks/section >}} {{% blocks/lead color="yellow" %}} # DESIGN-INTERVENTIONEN ----

Die Design­-Interventionen haben zum Ziel, Pflanzen und Tieren eine Stimme zu geben, ökologische Wissensbestände und Erfahrungen zu erweitern, neue Formen Spezies übergreifenden Zusammenlebens zu gestalten und dadurch ökologische Werte und Verhaltensweisen zu kultivieren. Dazu wird ein Internet of Things Toolkit entwickelt, das im Rahmen der gestalterischen Interventionen als medientechnologische Infrastruktur dient.
